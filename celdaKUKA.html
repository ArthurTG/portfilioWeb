Celda de Entrenamiento KUKA

Proyecto realizado en Unity utilizando Realidad Virtual para simular y capacitar personal industrial para operar un robot "KUKA" para la empresa " Integra Automation".

Sipnosis

El juego toma lugar en un bosque durante la noche, donde por razones que se desconocen el jugador se encuentra atrapado y tiene que explorar el mapa para poder encontrar las pistas necesarias y así poder escapar, pero tambien hay cierto peligro del cual tiene que escapar.

Al comienzo del juego corre una pequeña cinematica en la cual se muestran unos obstaculos donde se ve lo que parece ser una linterna. Esta herramienta sirve para iluminar las partes oscuras del mapa donde se necesite tener una mejor visión para encontrar las esferas y escapar de las criaturas que merodean el bosque.

Si bien el juego fue creado para un proyecto escolar y actualmente se encuentra detenido el desarrollo del mismo se estima que puede ser un buen elemento a considerar para un futuro proyecto llevando el alcance a otro nivel y mejorando la calidad.







Elementos Utilizados

  • Unreal Engine 4.
  • Blender.
  • Substance Painter.
  • Modelos 3D descargados de la pagina Mixamo (personajes).
  • Modelos 3D descargados de la página Free3D.
